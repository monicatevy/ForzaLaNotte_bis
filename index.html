Name Input

Bienvenue dans le jeu de mémoire pour saisir votre nom !

Voici comment jouer :

  • Trouvez les paires de lettres pour remplir votre nom.
  • Si vous faites une erreur, les cartes se cacheront de nouveau.
  • Cliquer sur envoyer réinitialisera votre entrée.
  • Vous pouvez cliquer sur le bouton "Remélanger les cartes" pour réorganiser les cartes pour avoir une lettre déjà saisie (par exemple pour alexa, on cherchera les paires A-A, L-L, E-E, X-X, et puis remélanger pour avoir A-A de nouveau
